Pen: CSS Scroll Down Button

by Jessica Angelone

In this pen the author has added 10 scroll down buttons each with their own background image. The author has also added various CSS to transform the page;
* webkit-transform: to make the images more 3D
* a span: to span across a wide area
* box-shadow: to add a shadow to the sections

See the Pen Demo: CSS scroll down button by Naoya (@nxworld) on CodePen.
